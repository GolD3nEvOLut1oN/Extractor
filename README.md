## ***Features***
- Dinamically gets all categories
- Navigate through the website pages
- Retrieve all products from each page
- From each product gets:
    - Name
    - URL
    - Sub category & its URL
    - Main category & its URL
    - Normal price
    - Internet price
    - Card price (usually the best price)
    - Product URL

## ***Requirements***

- Python 3.6.2
- Pip 3
- Scrapy 1.5.2
- Ubuntu 16.4 LTS
